# 常用命令

### jekyll 命令
```sh
# 构建静态站点
bundle exec jekyll build

# 构建并启动 jekyll 服务
bundle exec jekyll serve

# 清理构建结果
bundle exec jekyll clean
```

### gem 命令

```sh
# 更新所有已安装的包
gem update
```
